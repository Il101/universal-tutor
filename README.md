AI-powered learning platform for **any subject**, not only languages.

> Inspired by OpenLingo architecture, significantly adapted and extended for universal learning workflows.

## What I changed

- Reoriented product from language-only tutoring to **general-purpose learning**
- Expanded AI tool workflows for custom unit generation across domains
- Added and refined tool-calling patterns for structured study sessions
- Extended content handling to support broader educational use cases
- Integrated my own product direction, UX, and learning flow improvements

## Core Features

- AI tutor with context-aware conversations
- Tool-driven lesson/unit generation
- Spaced repetition workflow support
- Interactive exercises
- Reading and content transformation utilities
- User progress and learning memory

## Tech Stack

- Next.js + React + TypeScript
- PostgreSQL (Drizzle)
- Better Auth
- Vercel AI SDK
- Tailwind CSS

## Quick Start

```bash
git clone https://github.com/Il101/universal-tutor.git
cd universal-tutor
cp example.env.local .env.local
# fill env values
npm install
npm run dev
```
